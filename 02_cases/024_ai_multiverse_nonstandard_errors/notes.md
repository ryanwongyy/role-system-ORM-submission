# Case 24: AI multiverse / nonstandard errors in autonomous empirical analysis

## Summary

Run many autonomous agents on the same data and hypothesis to estimate nonstandard errors from model and sampling variation.

## Why it matters for typology

It makes latent fork choices visible and quantifiable, which is rare in human-only workflows.

## Caveats

- Falsification test: AI agents converge naturally to the same defensible result without multiverse management
- Main confounds: Tasks may overstate ambiguity if the research question is intentionally underspecified
- Evidence status: Preprint
