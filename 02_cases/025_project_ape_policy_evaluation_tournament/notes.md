# Case 25: Project APE end-to-end autonomous policy evaluation tournament

## Summary

Use APE's public papers, matches, and audit materials to study what human roles remain when an agentic system writes empirical economics papers end-to-end.

## Why it matters for typology

Project APE is the strongest real-world pressure test of the whole typology because it tries to automate the full workflow in a social-science domain.

## Caveats

- Falsification test: APE outputs prove reliable enough that human boundary-setting and accountability no longer materially change use decisions
- Main confounds: APE's tournament is an internal quality proxy and may not map to real journal or policy standards
- Evidence status: Public project / live website
