# Case 26: Project APE autonomy gradient: initialization, course correction, and revision mode

## Summary

Exploit APE's own autonomy framing to compare minimal prompts, guided prompts, optional course correction, and human-triggered revision as distinct human intervention points.

## Why it matters for typology

APE explicitly documents where humans still intervene, which most autonomous-research systems do not.

## Caveats

- Falsification test: Minimal-prompt and heavily guided outputs converge in quality and content
- Main confounds: Internal project design choices may not generalize to other autonomous-research systems
- Evidence status: Public project / live website
