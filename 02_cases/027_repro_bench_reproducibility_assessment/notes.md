# Case 27: REPRO-Bench agentic reproducibility assessment of social-science papers

## Summary

Benchmark whether agents can assess computational reproducibility by aligning paper claims with code/data packages and reproduction reports.

## Why it matters for typology

It separates rerunning code from adjudicating whether a paper's claims are actually supported.

## Caveats

- Falsification test: Agents match human reproducibility judgments on real social-science packages
- Main confounds: Benchmark tasks still provide agents with unusually convenient access to code and data
- Evidence status: Peer-reviewed conference paper
