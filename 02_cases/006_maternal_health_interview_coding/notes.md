# Case 6: Maternal-health interview coding with generative AI

## Summary

Use semistructured maternal-health interviews to test what is gained and lost when GenAI performs inductive coding and theme extraction.

## Why it matters for typology

Useful counterpoint to political-text cases because the topic is applied, interview-based, and semi-structured.

## Caveats

- Falsification test: LLM coding captures nuance and downstream interpretive usefulness as well as human coding
- Main confounds: Applied health interviews may not generalize to more theory-laden or politically contested domains
- Evidence status: Peer-reviewed article

## Sources collected

- `C006_P1` (peer-reviewed article / open): Generative AI for thematic analysis in a maternal health study: coding semistructured interviews using large language models — https://pmc.ncbi.nlm.nih.gov/articles/PMC12083056/
