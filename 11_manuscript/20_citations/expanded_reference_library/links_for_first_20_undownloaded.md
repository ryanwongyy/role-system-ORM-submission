Following manifest order. For each reference: (a) DOI / official publisher link, (b) known OA mirror if one exists, (c) Google Scholar search URL as a fallback for the author to check for OA versions I may have missed.

**⚠ Note on OA deposit URLs.** Author homepage URLs (hermanaguinis.com, home.uchicago.edu/~aabbott/, etc.) are the *known deposit locations* for each author, but specific filenames sometimes change as sites are reorganized. When a deep-link URL below returns a 404, the reliable fallback is:

1. Visit the author's homepage (linked in each entry)
2. Find the "Publications" page
3. Search for the paper title
4. Download from the current filename

In-session curl downloads of the listed URLs failed for most of them (server redirects, anti-bot measures, or moved filenames). The author accessing these URLs through a browser will see different results — most are live OA mirrors, just not curl-friendly.
